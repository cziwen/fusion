# Fusion Chaos - 2.1 Player Control

## 2.1 玩家控制 (Player Control)

- **旋转 (环境)**: 
    - **陀螺仪**: 在移动端通过手机陀螺仪控制场景的旋转。
    - **键盘**: 使用 `Q`, `E` 或 `,`, `.` 键模拟环境旋转。
- **移动 (平移)**:
    - 当前版本不提供 `W/A/S/D` 或方向键平移。
    - 玩家核心保持在场景中心，通过环境旋转完成对齐操作。
- **核心**: 玩家初始为一个 20x20 的白色方块。玩家方块丛的角度保持锁定（始终相对于屏幕正向），不再受直接旋转控制，而是通过环境旋转来对齐。
