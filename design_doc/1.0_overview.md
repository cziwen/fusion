# Fusion Chaos - 1.0 Overview

## 1.0 游戏概述 (Overview)

Fusion Chaos 是一款基于 Flame 引擎开发的极简风格 Roguelike 游戏。玩家置身于无尽的“混沌”（纯黑背景）中，通过控制一个白色的核心方块，吸引并合并散落在空间中的其他白块来不断壮大自己。

### 1.1 核心概念
- **混沌 (The Chaos)**: 游戏的世界背景，纯黑色，象征着虚无。
- **环境旋转 (World Rotation)**: 玩家不再手动旋转自身，而是通过手机陀螺仪或键盘控制整个环境的旋转。电脑/Web 端保留 `Q/E` 与 `,/.` 的按键旋转。
- **玩家核心 (Player Core)**: 玩家核心固定在场景中心，不再通过 `W/A/S/D` 或方向键进行平移。
- **控制环 (The Control Zone)**: 玩家周围存在可视控制环，进入该区域的方块会与环境旋转同步，便于精准对齐。当前实现中控制环半径与引力半径保持一致。
- **自由方块漂移 (Free-square Drift)**: 所有未连接方块会以恒定缓慢速度持续朝玩家核心移动，进入引力逻辑后再参与锁位与吸附。
- **方块生成 (Timed Spawn)**: 当前以定时生成为主，可动态调整生成数量与频率。
- **融合与消除 (Fusion & Match)**: 玩家吸引游离方块进行融合，并通过特定的形状排布（如五连珠、金字塔）触发消除。
- **连通性 (Connectivity)**: 消除后，失去与核心连接的方块会重新变为游离状态。
